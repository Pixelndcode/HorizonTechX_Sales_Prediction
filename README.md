# Sales Prediction Using Python

## Objective
To predict sales based on advertising expenditure using Machine Learning.

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Dataset
Advertising Dataset

## Model Used
Linear Regression

## Analysis Performed
- Data Loading and Cleaning
- Exploratory Data Analysis
- Sales vs Advertising Visualization
- Sales Prediction using Machine Learning
- Model Evaluation

## Key Insights
- TV advertising has a strong impact on sales.
- Higher advertising spend generally increases sales.
- Linear Regression successfully predicted sales values.
- The model achieved good prediction performance.

## Result
A sales prediction model was built to estimate future sales based on advertising investments.
